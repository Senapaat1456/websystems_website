How this works

GGPlot2 is a package in R that takes in dataframes and generates professional looking graphs. By entering the relevant data, this application will generate the code need to make the graph in R. Make sure for the dataframe and variable fields, you enter the exact name that R has stored in memory. Assuming everything is entered correctly, you can copy and paste the output directly into R Studio and when you run your code it will generate and display the graph. I recommend spiting the code that generates the dataframe and the code that graphs the data into two separate chunks. To learn more about chunks and R Notebooks, go to Installing R Studio. For more information about GGPlot2 and using it yourself without this tool go to ggplot.tidyverse.org Important note: The graph displayed in the browser is just an example of the selected style and has no relation to the actual data you are graphing.
